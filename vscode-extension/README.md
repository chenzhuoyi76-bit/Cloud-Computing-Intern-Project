# Cloud CI/CD Assistant Extension

## Usage

1. Start the Flask backend at `http://127.0.0.1:5000`
2. Open this `vscode-extension` folder in VS Code
3. Press `F5` to launch the Extension Development Host
4. Run `Cloud CI/CD: Ask Assistant` for intent recognition only
5. Run `Cloud CI/CD: Execute Task` to collect repo/deploy inputs and call the unified execution API
